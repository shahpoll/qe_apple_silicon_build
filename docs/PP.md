# Pseudopotentials
- Use PSLibrary or SSSP. Place UPFs under `cases/common/pp`. Example: Si.pbe-n-rrkjus_psl.1.0.0.UPF.
- Recommended cutoffs (30/240 Ry, dual = 8) follow the [SSSP Efficiency 1.3.0 table](https://www.materialscloud.org/discover/sssp/table/efficiency) entry for silicon.
- See QE PP portal for formats and sources.
